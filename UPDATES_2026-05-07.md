# Right Stay Site Updates - 2026-05-07

## Overview
This document summarizes the implementation work completed today across dynamic pricing, PriceLabs integration, booking UX, and admin tooling.

## 1) PriceLabs Integration (Read-Only Pull)

### Added
- New PriceLabs API client:
  - `lib/pricelabs.ts`
- New admin sync endpoint:
  - `app/api/admin/pricing/sync-pricelabs/route.ts`
- New PriceLabs mapping migration/table:
  - `supabase/migrations/add_property_pricelabs_mapping.sql`
  - Table: `property_pricelabs_mapping`

### What it does
- Maps each property to PriceLabs `listing_id + pms`.
- Pulls date-level pricing from PriceLabs.
- Stores synced prices in `property_daily_prices`.
- Tracks sync metadata (`last_synced_at`, `last_sync_status`, `last_sync_error`).
